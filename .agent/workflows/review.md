# Code Review Pipeline

Використовуйте цей workflow для проведення рев'ю коду:

1. **SOLID Перевірка**:
   - Чи порушує контролер, сервіс або репозиторій принципи SOLID (зокрема Single Responsibility).
2. **Memory Leaks у графах**:
   - Аналіз LangGraph нод на предмет безмежного накопичення даних у стейті або безкінечних циклів без fallback-механізмів.
3. **NestJS DI Валідація**:
   - Перевірка коректності імпортування модулів та провайдерів.
   - Чи відсутні циклічні залежності в ін'єкціях.
