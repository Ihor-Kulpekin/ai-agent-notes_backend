# Documentation Workflow

Виконуйте генерацію документації за наступними правилами:

1. **JSDoc Генерація**:
   - Згенерувати автоматичні JSDoc коментарі до складних методів.
   - Особлива увага для **LangGraph нод**: чітко описувати вхідні дані стейту, мутації всередині та вихідні дані.
2. **Swagger / OpenAPI**:
   - Оновити NestJS-контролери відповідними декораторами (`@ApiOperation`, `@ApiResponse`, `@ApiTags`, `@ApiProperty` в DTO).
