# TypeScript Style
- **Strict mode**: Завжди увімкнений.
- **Ніяких `any` або `unknown`** без крайньої необхідності.
- **Типізація об'єктів**: Використовувати `interface` для об'єктів, `type` — для union/intersection типів.
- **Методи масивів**: Надавати перевагу функціональним методам (`map`, `filter`, `reduce`) замість імперативного `for`.
- **Повернені значення**: Обов'язкова явна типізація повернених значень для всіх функцій та методів.
